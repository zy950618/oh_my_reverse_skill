# Multi-site Adapter Lab

This Phase 4 lab is a local synthetic evidence package. It documents solver/parity/drift behavior without live-site credentials or production traffic.

## Files

- `manifest.json`
- `negative_cases.json`
- `validation_report.json`
- `reports/replay_or_trace_report.md`

## Coverage

- site profile
- endpoint schema
- sign strategy
- token strategy
- captcha strategy
- fingerprint strategy
- pure API adapter
- FastAPI adapter
- SDK example
- replay report

## Validation

- `tools/validate_multi_site_adapter_lab.py`
